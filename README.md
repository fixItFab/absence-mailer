# Absence-Mailer

## Preparation

1. `npm i`
2. Create a `.env` file (`cp ./.env.template ./.env`) and enter your data

## NPM Scripts

- `npm start`
- `npm run serve`
- `npm run build`
- `npm run build-watch`
